# self-calibration
Explicit behavioral calibration and correction capture for OpenClaw. Replaces vague "self-improvement" with deterministic rule hardening.

Self-Calibration
A closed-loop improvement system driven entirely by user correction and concrete outcome-vs-intent mismatches. It prevents repeat mistakes without introducing background evolution or autonomous persona drift.

Core Mechanics:

Explicit Triggers: Operates strictly on user feedback and clear execution failures, explicitly ignoring silence, one-off instructions, and brainstorming.

Escalation Ladder: Evaluates whether a correction is a temporary fix (daily note), a durable preference (topic file), or requires a hard escalation to patch core policy files.

Conflict Resolution: Resolves rule contradictions deterministically (most specific rule wins; most recent rule wins).
